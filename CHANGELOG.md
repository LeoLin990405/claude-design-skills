# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-03-16

### Added

- **algorithmic-art** &mdash; p5.js generative art with seeded randomness and interactive parameter exploration
- **canvas-design** &mdash; Visual art and poster design with 30+ bundled fonts
- **frontend-design** &mdash; Production-grade frontend interface design
- **web-artifacts-builder** &mdash; Multi-component Claude.ai artifact scaffolding (React, Tailwind, shadcn/ui)
- **theme-factory** &mdash; 10 curated color/font themes with on-the-fly theme generation
- **slack-gif-creator** &mdash; Animated GIF creation optimized for Slack
- **brand-guidelines** &mdash; Anthropic brand colors and typography
- Repository structure with SKILL.md index, LICENSE, and CI workflow
- CONTRIBUTING.md and issue/PR templates
